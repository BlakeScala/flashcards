Flashcards

JavaScript (JS)

Programming language of HTML that allows users to interact with webpages.

Operators

mathematical symbols used in js to compare, assign and perform arithmetic operations on numbers and variables.

Variables

containers for storing data values.

Variable naming conventions

var x = definition ; (omit var for global assignment).

Functions

a set of statements stored under a variable name.

Methods

actions that can be performed on objects.

Arguments

specific values passed to functions through parameters.

Parameters

generic variables to be replaced by arguments in a function’s definition

Return

stops execution of a function and returns a value from that function.

Chaining methods

using more than one function on an object in a single line.

Strings

an immutable series of characters.

Booleans

a True or False value.

Undefined

primitive value given to variables before they have been assigned a value.

NaN

“Not-a-Number” returned when math functions fail.

Escape

a symbol used to indicate that a special character is not being used.

Alert

a pop-up command to send a message in a text box.

Comments

code that is not interpreted when the program is run. Used for programmer notes.

jQuery

an extensive library of javascript methods.

Attributes

modifiers for html elements.
